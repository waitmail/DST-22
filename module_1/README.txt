Задача проекта состояла в том чтобы скачать датасет,ноутбук шаблон, 
ответить на вопросы в викторине, а решения поместить в ноутбук шаблон.
В викторине набрал 27 баллов по одному баллу за каждый верно отвеченный вопрос. 
Еще предстоит поработать над lambda функциями.
Главным результатом данного модуля является решение размещенное в ноутбук шаблоне.
Доволен тем, что справился без ошибок с викториной.
При решении Не сильно использовал функции, над этим стоит еще поработать.
В этом модуле было интересно познакомится с библитекой pandas, с группировкой и обработкой данных.
Главным результатом этого проекта для является более лучшее понимание методов работы с данными.
Стратегию изучения последующих модулей менять не планирую.
